üñ•Ô∏è Desktop Player

The desktop player is an Electron application that runs on your display devices.

Features:

  • Real-time content synchronization
  • Automatic playlist playback
  • Support for images and videos
  • Force stop functionality
  • Connection status monitoring

Installation:

  1. Download the player from the releases page
  2. Install on your display device
  3. Enter the device code generated from the CMS
  4. The player will automatically connect and start displaying content
